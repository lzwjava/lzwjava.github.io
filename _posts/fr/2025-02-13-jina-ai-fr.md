## Jina AI

Ce script Python interagit avec les services Jina AI en utilisant des clés API et des arguments en ligne de commande. Il prend en charge deux tâches principales : la récupération du contenu d'une URL et l'exécution d'une requête de recherche. Le script récupère la clé API Jina à partir des variables d'environnement, garantissant un accès sécurisé aux services. Il utilise la bibliothèque `requests` pour effectuer des requêtes HTTP et `base64` pour décoder la requête de recherche. Le script affiche ensuite la réponse du service Jina AI.

```python
import os
import requests
from dotenv import load_dotenv
import argparse
import base64

load_dotenv()

api_key = os.environ.get("JINA_API_KEY")
if not api_key:
    raise ValueError("La variable d'environnement JINA_API_KEY n'est pas définie.")

parser = argparse.ArgumentParser()
parser.add_argument("--job", type=str, choices=['url', 'search'], help="Tâche à exécuter (url ou search)", required=True)
parser.add_argument("--input", type=str, help="Entrée pour la tâche", required=True)
args = parser.parse_args()

if args.job == 'url':
    url = f'https://r.jina.ai/{args.input}'
    headers = {'Authorization': f'Bearer {api_key}'}
    print(f"URL: {url}")
    print(f"En-têtes: {headers}")
    response = requests.get(url, headers=headers)
    print(response.text)

elif args.job == 'search':
    question = base64.b64decode(args.input).decode('utf-8', errors='ignore')
    url = f'https://s.jina.ai/{question}'
    headers = {
        'Authorization': f'Bearer {api_key}',
        'X-Engine': 'direct',
        'X-Retain-Images': 'none'
    }
    print(f"URL: {url}")
    print(f"En-têtes: {headers}")
    response = requests.get(url, headers=headers)
    print(response.text)

else:
    print("Veuillez spécifier --job url ou --job search")
```

## Tavily AI

[Tavily](https://tavily.com) est une API de recherche IA conçue spécifiquement pour les applications LLM. Elle fournit des résultats de recherche hautement pertinents en combinant la recherche web avec le traitement IA.

Pour utiliser Tavily, vous devrez :

1. Vous inscrire pour obtenir une clé API sur [tavily.com](https://tavily.com)
2. Installer le package Python.

```python
import os
from tavily import TavilyClient

# Récupérer la clé API depuis la variable d'environnement
TAVILY_API_KEY = os.getenv('TAVILY_API_KEY')

if TAVILY_API_KEY is None:
    raise ValueError("Clé API introuvable. Veuillez définir la variable d'environnement TAVILY_API_KEY.")

# Initialiser le TavilyClient avec la clé API récupérée
tavily_client = TavilyClient(api_key=TAVILY_API_KEY)

# Effectuer une requête de recherche
response = tavily_client.search("Qui est Leo Messi?")

# Afficher la réponse
print(response)
```
