### Jina AI

Ce script Python interagit avec les services Jina AI en utilisant des clés API et des arguments de la ligne de commande. Il prend en charge deux principaux travaux : le récupération de contenu à partir d'une URL et la réalisation d'une requête de recherche. Le script récupère la clé API Jina à partir des variables d'environnement, garantissant un accès sécurisé aux services. Il utilise la bibliothèque `requests` pour effectuer des requêtes HTTP et `base64` pour décoder la requête de recherche. Le script imprim alors la réponse du service Jina AI.

```python
import os
import requests
from dotenv import load_dotenv
import argparse
import base64

load_dotenv()

api_key = os.environ.get("JINA_API_KEY")
if not api_key:
    raise ValueError("La variable d'environnement JINA_API_KEY n'est pas définie.")

parser = argparse.ArgumentParser()
parser.add_argument("--travail", type=str, choices=['url', 'recherche'], help="Travail à exécuter (url ou recherche)", required=True)
parser.add_argument("--entree", type=str, help="Entrée pour le travail", required=True)
args = parser.parse_args()

if args.travail == 'url':
    url = f'https://r.jina.ai/{args.entree}'
    headers = {'Authorization': f'Bearer {api_key}'}
    print(f"URL: {url}")
    print(f"En-têtes: {headers}")
    response = requests.get(url, headers=headers)
    print(response.text)

elif args.travail == 'recherche':
    question = base64.b64decode(args.entree).decode('utf-8', errors='ignore')
    url = f'https://s.jina.ai/{question}'
    headers = {
        'Authorization': f'Bearer {api_key}',
        'X-Engine': 'direct',
        'X-Retain-Images': 'none'
    }
    print(f"URL: {url}")
    print(f"En-têtes: {headers}")
    response = requests.get(url, headers=headers)
    print(response.text)

else:
    print("Veuillez spécifier --travail url ou --travail recherche")

```

## Tavily AI

[Tavily](https://tavily.com) est une API de recherche AI conçue spécifiquement pour les applications de LLM. Elle fournit des résultats de recherche très pertinents en combinant la recherche web avec le traitement AI.

Pour utiliser Tavily, vous devez :

1. Vous inscrire pour obtenir une clé API sur [tavily.com](https://tavily.com)
2. Installer le package Python.


```python
import os
from tavily import TavilyClient

# Récupérez la clé API à partir de la variable d'environnement
TAVILY_API_KEY = os.getenv('TAVILY_API_KEY')

if TAVILY_API_KEY is None:
    raise ValueError("Clé API non trouvée. Veuillez définir la variable d'environnement TAVILY_API_KEY.")

# Initialisez le TavilyClient avec la clé API récupérée
tavily_client = TavilyClient(api_key=TAVILY_API_KEY)

# Effectuez une requête de recherche
response = tavily_client.search("Qui est Leo Messi?")

# Imprimez la réponse
print(response)

```
