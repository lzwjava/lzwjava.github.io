## Problèmes avec un ancien modem

Dans la maison de mes parents, le modem est assez vieux, probablement une dizaine d'années. La configuration réseau initiale était :

modem -> 3m sans fil -> TP-Link AX3000 (mode pont sans fil) -> 2m, un mur, sans fil -> Ordinateur portable

Cela entraînait une faible vitesse de téléchargement, avec un résultat Speedtest de seulement 10 Mbps.

Une configuration améliorée impliquait l'utilisation d'une connexion filaire :

modem -> câble de 2m -> TP-Link AX3000 (mode pont filaire) -> 4m sans fil, un mur -> Ordinateur portable

Cela a amélioré la vitesse de téléchargement jusqu'à 90 Mbps.

## Performances du nouveau modem

Dans ma propre maison, le modem est neuf, et le routeur TP-Link fonctionne bien en mode pont sans fil. La configuration réseau est :

modem -> 4m sans fil -> TP-Link AX3000 (mode pont sans fil) -> 2m sans fil -> Ordinateur portable

La qualité du réseau est bonne.


## Conseils de dépannage

Il n'y a pas de solution unique pour améliorer la vitesse Wi-Fi. Une bonne approche consiste à utiliser un câble pour tester chaque partie de votre réseau afin d'identifier les goulots d'étranglement. Comparez les vitesses lorsque vous utilisez une connexion filaire par rapport au Wi-Fi. Essayez également de connecter les appareils directement avec un câble pour voir si cela améliore les performances.
