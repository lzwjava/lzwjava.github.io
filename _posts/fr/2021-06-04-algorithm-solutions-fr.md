Chaque solution est organisée par numéro de problème et inclut le code source, les fichiers d'entrée, et parfois, une documentation supplémentaire.

## Structure du dépôt

- `README.md` : Ce fichier.
- `codeforces/` : Contient des solutions aux problèmes de Codeforces.
- `hacker_rank/` : Contient des solutions aux problèmes SQL de HackerRank.
- `leetcode/` : Contient des solutions aux problèmes de LeetCode.
- `lintcode/` : Contient des solutions aux problèmes de Lintcode.
- `nowcoder/` : Contient des solutions aux problèmes de Nowcoder.
- `uva/` : Contient des solutions aux problèmes de UVa.

### Codeforces

Chaque sous-répertoire dans le répertoire `codeforces` porte le nom d'un numéro de problème et inclut ce qui suit :

- `1.in` : Fichier d'entrée pour le problème.
- `src/Main.java` : Fichier de code source Java contenant la solution au problème.

### HackerRank

Le répertoire `hacker_rank` contient des fichiers SQL pour divers problèmes. Chaque fichier porte le nom du problème qu'il résout.

### LeetCode

Le répertoire `leetcode` contient des scripts Python pour différents problèmes. Les fichiers portent le nom des problèmes qu'ils résolvent, comme `two_sum.py` ou `valid_parentheses.py`.

### Lintcode

Chaque sous-répertoire dans le répertoire `lintcode` porte le nom d'un numéro de problème et inclut ce qui suit :

- `src/com/lintcode/Main.java` : Fichier de classe principal pour la solution.
- `src/com/lintcode/Solution.java` : Fichier de classe de solution pour le problème.
- `src/com/lintcode/` : Fichiers de classes supplémentaires selon les besoins du problème.

### Nowcoder

Chaque sous-répertoire dans le répertoire `nowcoder` porte le nom d'un problème et inclut ce qui suit :

- `Solution.java` : Fichier de classe de solution pour le problème.
- `Test.java` : Fichier de classe de test pour le problème.
- `TreeNode.java` : (Si applicable) Fichier de classe TreeNode pour les problèmes impliquant des structures d'arbre.
