## जिना एआई

यह पायथन स्क्रिप्ट जिना एआई सेवा के साथ एपीआई कुंजियों और कमांड-लाइन तर्कों का उपयोग करके इंटरैक्ट करती है। यह दो मुख्य कार्य समर्थित करती है: यूआरएल से सामग्री प्राप्त करना और खोज क्वेरी निष्पादित करना। स्क्रिप्ट पर्यावरण चर से जिना एपीआई कुंजी प्राप्त करती है, जिससे सेवा तक सुरक्षित पहुंच सुनिश्चित होती है। यह HTTP अनुरोधों के लिए `requests` लाइब्रेरी और खोज क्वेरी को डिकोड करने के लिए `base64` का उपयोग करती है। इसके बाद, स्क्रिप्ट जिना एआई सेवा से प्राप्त प्रतिक्रिया को प्रिंट करती है।

```python
import os
import requests
from dotenv import load_dotenv
import argparse
import base64

load_dotenv()

api_key = os.environ.get("JINA_API_KEY")
if not api_key:
    raise ValueError("JINA_API_KEY पर्यावरण चर सेट नहीं है।")

parser = argparse.ArgumentParser()
parser.add_argument("--job", type=str, choices=['url', 'search'], help="निष्पादित करने का कार्य (url या search)", required=True)
parser.add_argument("--input", type=str, help="कार्य के लिए इनपुट", required=True)
args = parser.parse_args()

if args.job == 'url':
    url = f'https://r.jina.ai/{args.input}'
    headers = {'Authorization': f'Bearer {api_key}'}
    print(f"यूआरएल: {url}")
    print(f"हेडर: {headers}")
    response = requests.get(url, headers=headers)
    print(response.text)

elif args.job == 'search':
    question = base64.b64decode(args.input).decode('utf-8', errors='ignore')
    url = f'https://s.jina.ai/{question}'
    headers = {
        'Authorization': f'Bearer {api_key}',
        'X-Engine': 'direct',
        'X-Retain-Images': 'none'
    }
    print(f"यूआरएल: {url}")
    print(f"हेडर: {headers}")
    response = requests.get(url, headers=headers)
    print(response.text)

else:
    print("कृपया --job url या --job search निर्दिष्ट करें।")
```

## टैविली एआई

[टैविली](https://tavily.com) एलएलएम अनुप्रयोगों के लिए विशेष रूप से डिज़ाइन की गई एक एआई खोज एपीआई है। यह वेब खोज को एआई प्रोसेसिंग के साथ मिलाकर अत्यधिक प्रासंगिक खोज परिणाम प्रदान करती है।

टैविली का उपयोग करने के लिए, आपको चाहिए:

1. [tavily.com](https://tavily.com) पर एपीआई कुंजी के लिए साइन अप करें।
2. पायथन पैकेज स्थापित करें।

```python
import os
from tavily import TavilyClient

# पर्यावरण चर से एपीआई कुंजी प्राप्त करें
TAVILY_API_KEY = os.getenv('TAVILY_API_KEY')

if TAVILY_API_KEY is None:
    raise ValueError("एपीआई कुंजी नहीं मिली। कृपया TAVILY_API_KEY पर्यावरण चर सेट करें।")

# प्राप्त एपीआई कुंजी के साथ TavilyClient आरंभ करें
tavily_client = TavilyClient(api_key=TAVILY_API_KEY)

# एक खोज अनुरोध करें
response = tavily_client.search("लियोनेल मेसी कौन हैं?")

# प्रतिक्रिया प्रिंट करें
print(response)
```

## ओपनराउटर का उपयोग

- चीन से जारी वीज़ा क्रेडिट कार्ड स्वीकार किए जाते हैं।

- आप देख सकते हैं कि लोग एलएलएम मॉडल का उपयोग रोलप्ले, प्रोग्रामिंग, मार्केटिंग, मार्केटिंग/एसईओ, और तकनीक जैसे श्रेणियों में कैसे करते हैं।
- हांगकांग में उपयोगकर्ताओं के लिए एन्थ्रोपिक एपीआई तक पहुंचने के लिए अभी भी वीपीएन की आवश्यकता होती है।

## इलेवनलैब्स एआई

- अच्छा अनुभव रहा। मैंने सेवा का परीक्षण करने के लिए $5 USD खर्च किए।
- आवाज क्लोनिंग बहुत अच्छी है।
- ऑडियो नमूने नीचे दिए गए हैं:
- [अंग्रेज़ी](assets/audios/ElevenLabs.mp3)
- [जापानी](assets/audios/ElevenLabs_1.mp3)
- [स्पेनिश](assets/audios/ElevenLabs_2.mp3)

```python
import os
import requests
from dotenv import load_dotenv
import argparse
import re

load_dotenv()

api_key = os.environ.get("ELEVENLABS_API_KEY")
if not api_key:
    raise ValueError("ELEVENLABS_API_KEY पर्यावरण चर सेट नहीं है।")

parser = argparse.ArgumentParser()
parser.add_argument("--file", type=str, help="भाषण में परिवर्तित करने के लिए मार्कडाउन फ़ाइल", required=False)
parser.add_argument("--text", type=str, help="भाषण में परिवर्तित करने के लिए टेक्स्ट", required=False)
parser.add_argument("--output", type=str, help="आउटपुट फ़ाइल का नाम", required=True)
parser.add_argument("--voice_id", type=str, default="21m00Tcm4TlvDq8iK2G8", help="उपयोग करने के लिए आवाज आईडी")

args = parser.parse_args()

if args.file:
    try:
        with open(args.file, 'r') as f:
            content = f.read()
            # फ्रंट मैटर हटाएं
            content = re.sub(r'---.*?---', '', content, flags=re.DOTALL)
            text = content.strip()
    except FileNotFoundError:
        print(f"त्रुटि: फ़ाइल नहीं मिली: {args.file}")
        exit(1)
    except Exception as e:
        print(f"फ़ाइल पढ़ने में त्रुटि: {e}")
        exit(1)
elif args.text:
    text = args.text
else:
    print("त्रुटि: या तो --file या --text निर्दिष्ट किया जाना चाहिए।")
    exit(1)

url = f"https://api.elevenlabs.io/v1/text-to-speech/{args.voice_id}"

headers = {
  "Accept": "audio/mpeg",
  "Content-Type": "application/json",
  "xi-api-key": api_key
}

data = {
  "text": text,
  "model_id": "eleven_flash_v2_5",
  "voice_settings": {
    "stability": 0.5,
    "similarity_boost": 0.5
  }
}

response = requests.post(url, json=data, headers=headers)

if response.status_code == 200:
    with open(args.output, 'wb') as f:
        f.write(response.content)
    print(f"ऑडियो {args.output} में सहेजा गया।")
else:
    print(f"त्रुटि: {response.status_code} - {response.text}")
```
