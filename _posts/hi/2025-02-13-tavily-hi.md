[टाविली](https://tavily.com) एक AI खोज एपीआई है जो LLM एप्लीकेशन के लिए 특별 रूप से डिज़ाइन की गई है। यह AI प्रोसेसिंग को मिलाकर वेब खोज से अत्यंत रelasent खोज परिणाम प्रदान करती है।

टाविली का उपयोग करने के लिए आपको करना होगा:

1. [टाविली](https://tavily.com) पर एपीआई की कुंजी के लिए पंजीकृत होना
2. पाइथॉन पैकेज को इंस्टॉल करने का.


```python
import os
from tavily import TavilyClient

# एपआई कुंजी को पर्यावरणीय चर से प्राप्त करें
TAVILY_API_KEY = os.getenv('TAVILY_API_KEY')

if TAVILY_API_KEY is None:
    raise ValueError("एपीआई कुंजी नहीं मिली। कृपया TAVILY_API_KEY पर्यावरणीय चर को स्थापित करें။")

# एपीआई कुंजी से प्राप्त किए गए TavilyClient को इंस्टेंट करें
tavily_client = TavilyClient(api_key=TAVILY_API_KEY)

# खोज अनुरोध भेजें
response = tavily_client.search("लेओ मेस्सी कौन है?")

# जवाब छापें
print(response)

```
