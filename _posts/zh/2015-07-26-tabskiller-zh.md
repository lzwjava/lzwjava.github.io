# TabsKiller

这是一个 Chrome 插件，当你的浏览器因为太多标签页而变得杂乱时，它会自动关闭最老的标签页。永远告别杂乱无章的浏览器体验！

一个神奇的 Chrome 插件，当打开网页过多的时候，会自动关掉最老的网页，让浏览器保持清爽！

# 功能

1. 当标签页数量超过设定的限制时，自动关闭最老的标签页。
2. 根据你的偏好自定义最大标签页数量（x）。
3. 锁定特定的 URL 模式，确保满足这些模式的标签页在标签页过多时保持打开。

# 特性

1. 当打开的网页超过一定数量的时候，会自动关掉最老的网页。
2. 可以设置最大的标签数量。
3. 可以设置锁定规则，使得满足这个规则的网页不被关闭。

## 故事

我通常会在 Chrome 中打开很多标签页。所以我会按 Ctrl + W 来关闭很多标签页。不断重复。所以我想写一个扩展来解决我的问题。然后我发现了“Tab Wrangler”扩展，它会在 x 分钟内不活动时关闭标签页。我从中学习，并制作了一个扩展，当标签页超过 x 个时关闭最老的标签页。并且锁定一些 URL 匹配特定模式的标签页。它帮助我很多。我不再需要关心标签页。标签页不会太多。它让我的生活更轻松。希望你也会喜欢它。
