[Tavily](https://tavily.com) 是为LLM应用量身打造的AI搜索API，它通过结合网页搜索和AI处理提供高度相关的搜索结果。

使用Tavily需要：
1. 在[https://tavily.com](https://tavily.com)登记以获取API密钥。
2. 安装Python软件包。

```python
import os
from tavily import TavilyClient

# 从环境变量中获取API密钥
TAVILY_API_KEY = os.getenv('TAVILY_API_KEY')

if TAVILY_API_KEY is None:
    raise ValueError("API密钥未找到。请设置TAVILY_API_KEY环境变量。")

# 用获取的API密钥初始化TavilyClient
tavily_client = TavilyClient(api_key=TAVILY_API_KEY)

# 进行搜索请求
response = tavily_client.search("谁是梅西？")

# 打印响应
print(response)
```
