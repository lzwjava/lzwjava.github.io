[Tavily](https://tavily.com) 是专门为LLM应用设计的AI搜索API，通过结合网络搜索和AI处理提供高度相关的搜索结果。

要使用Tavily，您需 Lily:

1. 在[tavily.com](https://tavily.com)注册API密钥
2. 安装Python软件包。

```python
import os
from tavily import TavilyClient

# 从环境变量中检索API密钥
TAVILY_API_KEY = os.getenv('TAVILY_API_KEY')

if TAVILY_API_KEY is None:
    raise ValueError("无法找到API密钥。请设置TAVILY_API_KEY环境变量。")

# 用检索的API密钥初始化TavilyClient
tavily_client = TavilyClient(api_key=TAVILY_API_KEY)

# 发出搜索请求
response = tavily_client.search(" Leone Messi是谁？")

# 打印响应
print(response)
```
