[Tavily](https://tavily.com) ist eine AI-Such-API, die speziell für LLM-Anwendungen entwickelt wurde. Sie bietet sehr relevante Suchergebnisse, indem sie die Websuche mit AI-Verarbeitung kombiniert.

Um Tavily zu verwenden, müssen Sie:

1. Auf [tavily.com](https://tavily.com) ein API-Schlüssel anfordern
2. Das Python-Paket installieren.


```python
import os
from tavily import TavilyClient

# Abrufen des API-Schlüssels aus der Umgebungsvariablen
TAVILY_API_KEY = os.getenv('TAVILY_API_KEY')

if TAVILY_API_KEY is None:
    raise ValueError("API-Schlüssel nicht gefunden. Bitte die Umgebungsvariable TAVILY_API_KEY festlegen.")

# Initialisieren des TavilyClient mit dem abgerufenen API-Schlüssel
tavily_client = TavilyClient(api_key=TAVILY_API_KEY)

# Ausführen einer Suchanfrage
response = tavily_client.search("Who is Leo Messi?")

# Ausgeben der Antwort
print(response)

```
