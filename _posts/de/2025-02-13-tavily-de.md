[Tavily](https://tavily.com) ist eine AI-Such-API, die speziell für LLM-Anwendungen entwickelt wurde. Sie bietet durch die Kombination von Websuche mit AI-Verarbeitung sehr relevante Suchergebnisse.

Um Tavily zu nutzen, müssen Sie:

1. Auf [tavily.com](https://tavily.com) einen API-Schlüssel anfordern
2. Das Python-Paket installieren.


```python
import os
from tavily import TavilyClient

# Holen Sie den API-Schlüssel aus der Umgebungvariable
TAVILY_API_KEY = os.getenv('TAVILY_API_KEY')

if TAVILY_API_KEY is None:
    raise ValueError("API-Schlüssel nicht gefunden. Bitte geben Sie die Umgebungvariable TAVILY_API_KEY an.")

# Initialisieren des TavilyClient mit dem abgerufenen API-Schlüssel
tavily_client = TavilyClient(api_key=TAVILY_API_KEY)

# Stellen Sie eine Suchanfrage
response = tavily_client.search("Wer ist Leo Messi?")

# Geben Sie die Antwort aus
print(response)

```
