- Die Singular Value Decomposition kann zur Bildkompression verwendet werden.

- Die Determinante einer Matrix zeigt, wie sich die Fläche verändert, wenn die Matrix auf eine Form angewendet wird.

- Die Eigenwerte werden durch die Gleichung \\[ \det(\lambda I - A) = 0 \\] definiert.

- NumPy's `linalg`-Paket kann zur Untersuchung und Lösung von Problemen der Linearen Algebra verwendet werden.

- Verwenden Sie Transformations der Linearen Algebra in n-dimensionalem Raum, um Matrizen zu verstehen.

- Eine Matrix ist eine Gruppe von Zahlen, die in Zeilen und Spalten angeordnet sind.

- Die Gleichung \\[ Ax = B \\] hat eine Lösung, wenn \\[ \text{rank}(A) = \text{rank}([A|B]) \\].

- Der Rang einer Matrix ist die Anzahl der linear unabhängigen Zeilen (oder Spalten) in ihrer echelon-Form.

---

Welchen Nutzen hat der Rang-Begriff? Er zeigt uns, wie viele Zeilen Informationen während einer linearen Transformation behalten. Im Gegensatz dazu zählt die Nullität (Dimension des Nulldraums), wie viele Zeilen auf Null abgebildet werden.

Was ist der Unterschied zwischen einer Matrix und einem einfachen Zahlenarray? Ein Array hat keine inhärente Richtung, während eine Matrix dies tut.

Was ist der Unterschied zwischen einer Matrix und einem Vektor? Ein Vektor ist eine eindimensionale Spalte von Zahlen und hat eine Richtung.
