GitHub Desktop Version 3.4.13 (arm64) zeigt Probleme bei der Schriftdarstellung im Text-Diff-Panel für bestimmte Sprachen. Die betroffenen Sprachen umfassen:

- Chinesisch (sowohl vereinfacht als auch traditionell)
- Hindi
- Japanisch

Die folgenden Sprachen werden jedoch korrekt dargestellt:
- Englisch
- Spanisch
- Deutsch
- Arabisch
- Französisch
