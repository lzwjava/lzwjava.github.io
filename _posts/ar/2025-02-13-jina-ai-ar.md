## جينا للذكاء الاصطناعي

هذا السكريت المكتوب بلغة بايثون يتفاعل مع خدمات جينا للذكاء الاصطناعي باستخدام مفاتيح واجهة برمجة التطبيقات وحجج سطر الأوامر. يدعم مهمتين رئيسيتين: جلب المحتوى من رابط وإجراء استعلام بحث. يسترجع السكريت مفتاح واجهة برمجة التطبيقات لجينا من متغيرات البيئة، مما يضمن الوصول الآمن للخدمات. يستخدم مكتبة `requests` لإجراء طلبات HTTP ومكتبة `base64` لفك تشفير استعلام البحث. بعدها، يطبَع الاستجابة من خدمة جينا للذكاء الاصطناعي.

```python
import os
import requests
from dotenv import load_dotenv
import argparse
import base64

load_dotenv()

api_key = os.environ.get("JINA_API_KEY")
if not api_key:
    raise ValueError("متغير البيئة JINA_API_KEY غير محدد.")

parser = argparse.ArgumentParser()
parser.add_argument("--job", type=str, choices=['url', 'search'], help="المهمة المراد تنفيذها (url أو search)", required=True)
parser.add_argument("--input", type=str, help="مدخلات المهمة", required=True)
args = parser.parse_args()

if args.job == 'url':
    url = f'https://r.jina.ai/{args.input}'
    headers = {'Authorization': f'Bearer {api_key}'}
    print(f"الرابط: {url}")
    print(f"الرؤوس: {headers}")
    response = requests.get(url, headers=headers)
    print(response.text)

elif args.job == 'search':
    question = base64.b64decode(args.input).decode('utf-8', errors='ignore')
    url = f'https://s.jina.ai/{question}'
    headers = {
        'Authorization': f'Bearer {api_key}',
        'X-Engine': 'direct',
        'X-Retain-Images': 'none'
    }
    print(f"الرابط: {url}")
    print(f"الرؤوس: {headers}")
    response = requests.get(url, headers=headers)
    print(response.text)

else:
    print("الرجاء تحديد --job url أو --job search")
```

## تافيلي للذكاء الاصطناعي

[تافيلي](https://tavily.com) هي واجهة برمجة تطبيقات بحث بالذكاء الاصطناعي مصممة خصيصًا لتطبيقات نماذج اللغة الكبيرة. توفر نتائج بحث عالية الصلة من خلال دمج بحث الويب مع معالجة الذكاء الاصطناعي.

لاستخدام تافيلي، ستحتاج إلى:

1. التسجيل للحصول على مفتاح واجهة برمجة التطبيقات في [tavily.com](https://tavily.com)
2. تثبيت الحزمة الخاصة بلغة بايثون.

```python
import os
from tavily import TavilyClient

# استرجاع مفتاح واجهة برمجة التطبيقات من متغير البيئة
TAVILY_API_KEY = os.getenv('TAVILY_API_KEY')

if TAVILY_API_KEY is None:
    raise ValueError("مفتاح واجهة برمجة التطبيقات غير موجود. الرجاء تعيين متغير البيئة TAVILY_API_KEY.")

# تهئية عميل تافيلي بمفتاح واجهة برمجة التطبيقات المسترجع
tavily_client = TavilyClient(api_key=TAVILY_API_KEY)

# إجراء طلب بحث
response = tavily_client.search("من هو ليونيل ميسي؟")

# طباعة الاستجابة
print(response)
```
