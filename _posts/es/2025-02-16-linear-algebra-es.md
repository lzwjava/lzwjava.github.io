* La descomposición de valores singular se puede utilizar para la compresión de imágenes.
* El determinante de una matriz indica cuánto cambia el área cuando se aplica la matriz a una forma.
* Los valores propios se definen por la ecuación \\[ \det(\lambda I - A) = 0 \\].
* El paquete `linalg` de NumPy se puede utilizar para estudiar y resolver problemas de álgebra lineal.
* Utilice transformaciones de álgebra lineal en un espacio n-dimensional para comprender matrices.
* Una matriz es un grupo de números organizados en filas y columnas.
* La ecuación \\[ Ax = B \\] tiene una solución cuando \\[ \text{rank}(A) = \text{rank}([A|B]) \\].
* El rango de una matriz es el número de filas (o columnas) linealmente independientes en su forma escalonada.
---
¿Cuál es el uso del concepto de rango? Nos dice cuántas filas retienen información durante una transformación lineal. A diferencia de la nulidad (dimensión del espacio nulo), que cuenta cuántas filas se mapean a cero.
¿Cuál es la diferencia entre una matriz y un array normal de números? Un array no tiene dirección inherentemente, mientras que una matriz sí.
¿Cuál es la diferencia entre una matriz y un vector? Un vector es una columna unidimensional de números y tiene una dirección.
