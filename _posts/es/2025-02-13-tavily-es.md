[Tavily](https://tavily.com) es una API de búsqueda de IA diseñada especialmente para aplicaciones de LLM (Large Language Model). Proporciona resultados de búsqueda altamente relevantes al combinar la búsqueda web con el procesamiento de IA.

Para utilizar Tavily, necesitarás:

1. Registrarte para obtener una clave de API en [tavily.com](https://tavily.com)
2. Instalar el paquete de Python.


```python
import os
from tavily import TavilyClient

# Recuperar la clave de API desde la variable de entorno
TAVILY_API_KEY = os.getenv('TAVILY_API_KEY')

if TAVILY_API_KEY is None:
    raise ValueError("Clave de API no encontrada. Por favor, establece la variable de entorno TAVILY_API_KEY.")

# Inicializar TavilyClient con la clave de API recuperada
tavily_client = TavilyClient(api_key=TAVILY_API_KEY)

# Realizar una solicitud de búsqueda
response = tavily_client.search("¿Quién es Leo Messi?")

# Imprime la respuesta
print(response)
```
