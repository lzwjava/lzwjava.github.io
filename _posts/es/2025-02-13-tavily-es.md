[Tavily](https://tavily.com) es una API de búsqueda de IA diseñada específicamente para aplicaciones LLM. Proporciona resultados de búsqueda altamente relevantes al combinar la búsqueda web con el procesamiento de IA.

Para usar Tavily, necesitarás:

1. Registrarte para obtener una clave de API en [tavily.com](https://tavily.com)
2. Instalar el paquete de Python.

```python
import os
from tavily import TavilyClient

# Recuperar la clave de API a partir de la variable de entorno
TAVILY_API_KEY = os.getenv('TAVILY_API_KEY')

if TAVILY_API_KEY is None:
    raise ValueError("No se ha encontrado la clave de API. Por favor, establezca la variable de entorno TAVILY_API_KEY.")

# Inicializar el TavilyClient con la clave de API recuperada
tavily_client = TavilyClient(api_key=TAVILY_API_KEY)

# Realizar una solicitud de búsqueda
response = tavily_client.search("Quién es Leo Messi?")

# Imprimir la respuesta
print(response)
```
