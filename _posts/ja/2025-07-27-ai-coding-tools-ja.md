最近、Claude Code を成功させて実行できたので、その過程で選んだツールについて共有したいと思います。また、過程でいくつかの [AI ツールのヒント](ai-tips-en) を収集しました。

正直、Claude Code にはかなり遅れてから取り組みました。

[Claude 3.5 Sonnet](https://www.anthropic.com/news/claude-3-5-sonnet) は2025年2月末頃にリリースされました。

正直、最近まで成功して試すことができませんでした。その理由の一つは、Anthropic API が中国のビザカードをサポートしていないことです。

もう一つの理由は、[Claude Code Router](https://github.com/musistudio/claude-code-router) が登場したことです。そのため、最近の試みが成功したのです。

Claude Code が素晴らしいと聞く話をよく聞きます。2025年7月にGemini CLIを試しました。コードを修正するように何度か会話した後、二度と使いませんでした。

また、別のソフトウェアエージェントであるAiderも試しました。Cursorは約6ヶ月使った後、ほとんど使わなくなりました。VSCodeの上に構築されているため、Cursorの多くのプラグインが動作しないことに気づいたからです。Clineもしばらく実験しましたが、広く使うことはありませんでした。

VSCodeプラグインでGrok 3ベータモデルを使用し、カスタマイズされたモデルでCopilotプラグインを使用しています。とても良い感じです。

Claude Code が私の習慣を変えるとは思いません。しかし、成功して実行できて、さらに数回実行する忍耐力があるので、今後数週間の感想を見てみましょう。

私は10年のソフトウェアエンジニアリングの経験を持つ厳選ユーザーです。そのため、ツールが実際の使用で優れていることを望みます。ブランドにはこだわりません。実際の日常的な有用性だけが気になります。
